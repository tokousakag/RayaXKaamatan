- 🌾 **Rice Stalk Animation** — Animasi padi bergoyang di bahagian bawah

- ⏳ **Countdown Timer** — Hitung mundur ke Aidiladha (27 Mei) & Kaamatan (30 Mei)
- 🎵 **4 Lagu Dusun dari YouTube** — DJ Remix Kaamatan 2026, Sumazau Pusing, Lagu Kaamatan, Virtual Choir

- 🏮 **Floating Lanterns** — Pelita melayang naik ke langit
- 📱 **Responsive** — Mobile & desktop friendly

## 🚀 Deploy ke GitHub & Vercel

### 1. Upload ke GitHub

```bash
# Clone atau init repo baru
git init
git add .
git commit -m "🌾🌙 Kaamatan & Aidiladha 2026 — Sabah Bersatu"
git branch -M main
git remote add origin https://github.com/USERNAME/kaamatan-aidiladha-2026.git
git push -u origin main
```

### 2. Deploy ke Vercel

**Cara 1 — Vercel Dashboard (Paling Mudah):**
1. Pergi ke [vercel.com](https://vercel.com) → Login
2. Klik **"New Project"**
3. Import dari GitHub repo anda
4. Klik **Deploy** — selesai! URL terus diberikan ✅

**Cara 2 — Vercel CLI:**
```bash
npm install -g vercel
vercel login
vercel --prod
```

### 3. GitHub Pages (Alternatif Percuma)

1. Pergi ke repo GitHub → **Settings** → **Pages**
2. Source: **Deploy from a branch** → pilih `main` → folder `/root`
3. URL: `https://USERNAME.github.io/kaamatan-aidiladha-2026/`
